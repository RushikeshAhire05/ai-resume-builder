# AI Resume & Portfolio Builder

This project is a simple AI-based web application designed to help students create professional resumes and basic portfolios easily.

Many students struggle with writing resumes, choosing the right words, and formatting their skills and projects properly. This application solves that problem by generating resume points automatically using AI and allowing users to download their resume as a PDF.

---

## Features

- Takes user details such as skills, education, projects, and target job role  
- Generates professional and role-focused resume points using AI  
- Displays a resume preview inside the application  
- Allows users to download the resume as a PDF file  
- Simple and user-friendly interface  

---

## Technologies Used

- Python  
- Streamlit (for frontend and UI)  
- Transformers (for AI text generation)  
- Scikit-learn (for basic keyword matching)  
- FPDF (for PDF generation)  
- Git & GitHub (for version control)

---

## How the System Works

1. The user enters personal details like skills, education, and projects.  
2. The system processes the input and sends it to an AI model.  
3. The AI generates professional resume bullet points.  
4. A resume preview is shown to the user.  
5. The user can download the generated resume as a PDF.
